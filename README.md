# 🎮 Game Hub

A real-time multiplayer party game platform for phones and a shared screen — host on a laptop or TV, players join from their phones via QR code. No app install needed.

## Games

### 👑 Empire ⚔️
A word-guessing party game.

1. Everyone secretly submits a word (optionally within a chosen category)
2. The shuffled list of words is revealed — but not who wrote them
3. Players take turns guessing which word belongs to which person — guess correctly and you "capture" them into your empire
4. **Last empire standing wins!**

### ❓ Trivia
Fast-paced multiple-choice trivia rounds with live scoring and reactions.

## How It Works

1. **The host** opens Game Hub on a laptop or TV screen and picks a game
2. **Players join** on their phones by scanning the QR code
3. Play together in real time

## Running Locally

```bash
npm install
npm start
```

Then open `http://localhost:3000` on the host device.
